# Sounds Directory

Place your ambient sound files here. Supported formats: .ogg (primary), .mp3 (fallback)

Expected files:
- rain.ogg / rain.mp3
- thunder.ogg / thunder.mp3
- wind.ogg / wind.mp3
- forest.ogg / forest.mp3
- ocean.ogg / ocean.mp3
- fire.ogg / fire.mp3
- cafe.ogg / cafe.mp3
- whitenoise.ogg / whitenoise.mp3

The app uses Web Audio API's fetch() + decodeAudioData() pipeline.
All files should be loopable ambient loops (ideally 30s–2min).
